Project summary

Power BI dashboard for monitoring Incident and Request Management KPIs across IT service operations. The dashboard provides executive-level KPIs, operational insights, agent performance cards, and tabular reports for analysis to drive SLA compliance and backlog reduction.

Dashboard pages (as in PDF)

Incident Management (PDF page 1)

Overview KPI cards, time-series, breach & SLA charts, category donut, backlog visual and assignment-group treemap. Example snapshot values visible on the PDF: Total Incidents = 5000; Closed = 1720; Backlog = 3280; Same-day closures = 186; First Contact Resolution ≈ 49%; Cumulative SLA ≈ 20.70%; Avg Resolution Time ≈ 174.83 hrs.
